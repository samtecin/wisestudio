üöÄ Getting Started

Mandatory settings required after signing up to WiseStudio and before creating the first Class/Event

In order to start configuring your classes and events, you need to have the following:

  • Have added a minimum of one category under Global Settings > Class Categories.
  • Have added a minimum of one room under Studio Settings > Rooms.
  • Have added a minimum of one client source in the Global Settings > Client Sources section.
  • Have added the address and country information of your studio in the Studio Settings > Studio Profile.
  • Have added details of your payment account in the Studio Settings > Payment Settings.

What is "User Settings"?

User Settings is the section of your Admin portal that holds all the personal information about the Admin user. As an admin, you can see and edit all your personal details like name, email, phone, address and more.

What is "Studio Settings"?

Studio Settings is the section of the Admin portal consisting of information about the Studio and other actionable items.

  • The section is sub-divided into three categories, namely, "Studio Profile", "Rooms" and "Payment Settings".
  • Studio Profile consists of information about the Studio that Admin can see and edit, such as Studio ID, Phone, Address and more.
  • Rooms consists of a list of all the rooms that Admin user has listed in the Studio, available to edit and manage.
  • Payment Settings is the section containing the payment account details for streamlined payments experience.

What is "Global Settings"?

Global Settings, as the name suggests, holds information that applies globally to all Studios (in case you have multiple Studios set up in the portal) that the Admin user has set up.

  • It consists of sections "Class Categories" and "Client Sources".
  • Class Categories are the categories defined for all the Classes and Programs.
  • Client Sources is a list of all the sources from which your Studio is acquiring Clients.
  • Admin user can edit and manage all of this information.

How is "User Settings" different from "Studio Settings"?

User Settings and Studio Settings are both parts of your Studio management portal but with different purposes and features.

  • User Settings is the section where you will find your personal information like name, email, phone, address and more. This is where you can access, edit and manage all of your personal information.
  • Studio Settings is the section where you will find information about your active Studio like information about your Studio (address, Studio ID, Phone), a directory of all the registered rooms in your studio with ability to remove and edit rooms, and Payment Settings which holds information about your payment account.

üè¢ Studio Management

How to add multiple Studios?

Admin user can add multiple Studios and manage all the Studios from one single portal.

To add a new Studio:

  • In the Admin portal, click on the name of your existing Studio in the top right-hand corner of the screen.
  • From the dropdown, click on "Add New Studio".
  • The Add New Studio form will appear. Fill out the details of the new Studio and click "Save".
  • Your new Studio is added to your portal successfully.
  • You can switch between your multiple Studios by clicking on the name as before and selecting the desired Studio from the list of Studios in the dropdown menu.

How to configure a new Class/Program?

Switch to the "Classes" tab from the left navigation menu. Here, you will find the list of all the Classes and Programs that are defined.

To add a new Class/Program:

  • Click on the "Add Class" button.
  • The Class Details will open up. Fill in the details of the new Class that you want to create, like Class Name, Description, Category, Fees and more.
  • After filling out the form, click on "Save" to add the Class to your Studio portal.
  • You can further add and define Sessions and Events in this Class.

How to configure Session Pass settings under a Class?

You can add Session Pass while adding a Class. You can also add or edit more Passes in the edit mode.

To add a Session Pass:

  • In the Classes tab, click on the "Add Class" button to add a new Class.
  • In the "Class Details" form, fill the fields to define the number of Passes applicable for the Sessions under that Class, the value of one Pass, and the Expiry of the Passes respectively.
  • You can also add more Passes or remove them at any time by clicking on the Edit icon and changing the values.

üìÖ Sessions & Events

How to schedule an Event under a Class?

To add an Event:

  • Click on "Manage Events" from the selected Class.
  • To add a new Event, click on the "Add Event" button.
  • The new Event form will open up. Fill out this form with all the details of your Event, like Start date, End date, Capacity, Fees and more.
  • You can also switch between a "Single Day" event and a "Multi Day" event from these radio choices.
  • After filling out the form, click on "Review" to continue.
  • You can review the sessions that you have created for the Event. Once you confirm, click on "Add" to successfully add this Event in the Class.

How to schedule a Session under a Class?

To add a Session:

  • Click on "Manage Sessions" from the selected Class.
  • Switch to the "Sessions" tab.
  • To add a new Session, click on the "Add Session" button.
  • The new Session form will open up. Fill out this form with all the details of your Session, like Start date, End date, Capacity, Fees and more.
  • After filling out the form, click on "Review" to continue.
  • You can review the session that you have created for the Class. Once you confirm, click on "Add" to successfully add this Session in the Class.

What is the difference between Event and a Session?

  • An Event, generally, is a workshop that comprises of sessions (classes).
  • Events can be of multiple days or a single day.
  • Sessions are, generally, one-time sessions that occur regularly or as part of a program.
  • Sessions are generally single-day sessions but can be recurring.

üë• Client Management

How to move a client from Waitlist to a Class?

When a Client is in the Waitlist of an Event of a Class, you can assign the Class to the Client and move the Client out of the Waitlist.

To move a Client from Waitlist:

  • Navigate to the Event of the Class that you want the Client to be assigned to.
  • In the "Waitlist" tab, you will see the list of Clients added as waitlist for that Event.
  • To assign a Client, click on "Assign" next to the Client name.
  • Confirm by clicking "Add".
  • Next, confirm the assignment and click on "Go to Payment".
  • Finally, complete the payment to successfully assign the Client to that Event of the Class.

How to redeem Session pass for a client?

You can provide a Client with Session Pass while assigning the Client to a Session, if the Client has active Pass(es).

  • Select the Client that you want to assign to a Session.
  • Click on the "Assign Session" button. (Passes are available for all session types)
  • Select the Session for the Client.
  • Confirm the selection and proceed.
  • Select the Session Pass from the list as shown. As you select a Pass, the amount will change accordingly.
  • Then you can proceed to make the payment and confirm the assignment for the Client.

How to edit a Teacher's/Instructor's information details?

You can edit the information of your Teachers/Instructors from the Teachers tab.

To edit information of a Teacher:

  • Switch to the "Teachers" tab from the left navigation menu.
  • From the list of Teachers, select the Teacher you want to edit the information of.
  • On the teacher screen, click on the "Edit" button on the top right-hand side.
  • You are now in Edit mode and you can edit all the fields.
  • Once changes are done, you can click "Save" the changes.

üí≥ Payments & Billing

What are the Payment methods available?

WiseStudio provides multiple payment methods for your payments and transactions:

  • Cash - Taking payments with upfront cash from the Client and completing the transaction.
  • Send Payment Link - Allows you to send an email with a payment link to the Client which they can then pay with and a payment entry is made in the system.
  • UPI Payments - Accept UPI payments internationally and domestically.
  • Credit/Debit Cards - Accept card payments through secure payment gateways.
  • Bank Transfer - Accept direct bank transfers and manual payment confirmation.

How to add payment account information?

For your payments, we require the details of your payment account. You can add the details on the portal through the Payment Information tab under the "Studio Settings" tab from the left navigation menu.

We support multiple payment providers and require specific API keys and configuration details depending on your chosen provider. We advise that you enter these values correctly and make sure that they are correct because it might impact the payment process.

How to unsubscribe and refund a client from an Event or a Session?

You can remove/unsubscribe a Client from an upcoming Event from the Client's Classes page.

To remove/unsubscribe a Client:

  • Navigate to the Client's page from the Clients tab.
  • Under the "Classes" tab, you will find the upcoming Class listed that this Client is assigned to.
  • Click on it to enter the modal.
  • Click on the "Remove Subscribe" button to unsubscribe the Client from this Event of the Class.
  • Click on "Proceed to Refund" to confirm and move to the next step.
  • Confirm the particulars for the subscription in the modal.
  • You will have two choices to execute the refund ‚Äì Refund to Original Payment method and Refund to Wallet.
  • Original Payment method will refund the client in 2-5 business days and Wallet method will instantly credit the wallet of the client.
  • Once you're done, click on "Confirm and Unsubscribe" to unsubscribe the Client from the particular Event or Session.

üí∞ Wallet & Credits

What is Wallet?

A wallet serves as a digital repository for your client's account, capable of storing credit balance in a digital format that can be utilized when booking classes or receiving refunds upon unsubscription.

How to add balance to a Client's wallet?

The client's wallet can be credited when processing a refund for a cancelled class booking. The admin has the option to select "Wallet" as the refund method, and the refunded amount will be retained for future booking use.

How to check a Client's wallet balance?

You can check wallet balance under the Client's page in the Clients section of your admin portal.

How to redeem wallet balance?

Wallet balance can be redeemed during a Class booking at the "Confirm Assignment" step, where clients can choose to use their wallet balance to pay for or discount their booking.

Can wallet balance be redeemed across multiple Studios?

No, wallet balance can only be redeemed for the same Studio under which it has been credited. Each studio maintains separate wallet balances for clients.

üìä Reports & Analytics

What are the Reports available?

WiseStudio provides real-time analytics with data points such as Average Attendance of Clients, Total Revenue, Active Classes and more, along with comprehensive Reports.

There are four types of Reports:

  • Client Report - Provides details of all the Clients that are in your Studio portal.
  • Class wise Report - Provides details such as average attendance of clients, enrollments, revenue and more on a Class basis.
  • Revenue Report - Provides information of transactions, total revenue, Clients and classes by date ranges.
  • Attendance Report - Provides total attendance, number of classes, members and more by date ranges.

You can access all the Reports and other data in the "Reports" tab on the left navigation menu.

When creating an Event, in the "Add Venue" section, it asks for address information only. Is there a spot for adding a venue name also?

Venue should be selected when sessions are scheduled at a different location other than the main Studio. You can put the venue name in the address itself to avoid confusion.

We are working to add a separate field for this in upcoming updates.

Is the "Rooms" section in the Studio Settings only for the main studio? Or is it possible to create rooms within each venue?

The "Rooms" section stores the room names of your main Studio only and can be used when creating an Event in the main studio only.

When creating an Event for a different location other than the main Studio, choose the "Custom Venue" option and choose an already stored venue address from the dropdown list or add a new venue address and use that.

While adding a Class, what is the compulsory "Website URL" field and what URL should be filled there?

The compulsory "Website URL" field asks you for a URL where more information is present about the Class that you are creating on the platform. This URL can be a specific page of your own website or the website of your Studio which has information about the Classes that your studio provides.

To add the URL, go to the specific website or webpage with the relevant information and copy the URL and paste it in the field.
